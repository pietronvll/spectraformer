SpectraFormer project is a transformer-based Machine Learning project aimed for Raman spectra unmixing for graphene buffer layer on SiC substrate.

Tips:

To use tensorboard, type in terminal "tensorboard --logdir=logs"

To use dashboard, type in terminal "streamlit run dashboard.py"

Useful command to watch gpu load in real time: "watch -n 1 nvidia-smi" where 1 is update time in seconds.
